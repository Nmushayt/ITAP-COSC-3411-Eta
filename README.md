# Study Mode: ALL Linux Commands

Command: whoami  
Description: Show current logged-in user.  
Example: $ whoami  
-----------------------------------------

Command: cd  
Description: Change directory.  
Example: $ cd /etc  
-----------------------------------------

Command: ls  
Description: List files and folders.  
Example: $ ls  
-----------------------------------------

Command: ls -l  
Description: Long listing format (permissions, owner, size).  
Example: $ ls -l  
-----------------------------------------

Command: ls -a  
Description: Show all files including hidden ones.  
Example: $ ls -a  
-----------------------------------------

Command: clear  
Description: Clear the terminal screen.  
Example: $ clear  
-----------------------------------------

Command: man  
Description: Show manual help pages.  
Example: $ man ls  
-----------------------------------------

Command: locate  
Description: Search the filesystem database.  
Example: $ locate passwd  
-----------------------------------------

Command: whereis  
Description: Locate binary, man page, and source.  
Example: $ whereis python  
-----------------------------------------

Command: echo  
Description: Print text or variables to screen.  
Example: $ echo Hello World  
-----------------------------------------

Command: find  
Description: Search filesystem by name, type, size, etc.  
Example: $ find / -name passwd  
-----------------------------------------

Command: cat  
Description: Display contents of a file.  
Example: $ cat file.txt  
-----------------------------------------

Command: touch  
Description: Create an empty file.  
Example: $ touch notes.txt  
-----------------------------------------

Command: mkdir  
Description: Create a new directory.  
Example: $ mkdir myfolder  
-----------------------------------------

Command: sudo  
Description: Run a command as root (superuser).  
Example: $ sudo apt update  
-----------------------------------------

Command: nl  
Description: Number lines in a file.  
Example: $ nl file.txt  
-----------------------------------------

Command: wc  
Description: Count lines, words, and characters.  
Example: $ wc -l file.txt  
-----------------------------------------

Command: ps  
Description: Show running processes.  
Example: $ ps  
-----------------------------------------

Command: top  
Description: Show real-time system resource usage.  
Example: $ top  
-----------------------------------------

Command: history  
Description: Show command history.  
Example: $ history  
-----------------------------------------

Command: grep  
Description: Search for text patterns in files.  
Example: $ grep hello file.txt  
-----------------------------------------

Command: cut  
Description: Cut text by delimiter and field.  
Example: $ cut -d: -f1 /etc/passwd  
-----------------------------------------

Command: chmod  
Description: Change file and directory permissions.  
Example: $ chmod 755 script.sh  
-----------------------------------------
